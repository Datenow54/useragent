Generate **realistic, rotating User Agents** for Chrome (v133-146), Firefox, and social media platforms (Facebook, Reddit, Instagram, Twitter, TikTok) across all devices: iOS, macOS, iPhone, iPad, Android, Windows.

## ✨ Features

- ✅ **Chrome versions 133 to 146** (latest)
- ✅ **Firefox versions 120-135**
- ✅ **All devices**: iPhone, iPad, Mac, Android phones/tablets, Windows
- ✅ **Social WebViews**: Facebook, Reddit, Instagram, Twitter, TikTok
- ✅ **Self-signup ready** - realistic fingerprints
- ✅ **No external dependencies**
- ✅ **CLI & Python API**
